INSTALL — DTC Friendly on Shopify Craft
================================================

Step 0, then two copy-paste blocks. ~5 minutes. Nothing else.

STEP 0 — GET THE FONTS
Follow ../../fonts/DOWNLOAD.txt. You must have the renamed .woff2
files uploaded to your theme's assets/ folder before the code below
will resolve. This is the ONLY step that can go wrong — the filenames
must match exactly. Get this right and the rest cannot fail.

STEP 1 — Paste the @font-face block
Open the bottom of assets/base.css in your Craft theme code editor.
Paste the entire contents of:  font-face.css

STEP 2 — Paste the CSS variable overrides
Directly below what you just pasted (same file), paste the entire
contents of:  css-variables.css
This retargets Craft's heading/body font variables directly. It
needs no Theme Editor setup and it survives theme updates. Craft
renders headings via selectors like .product__title, .testimonial__quote — they pick up the new
variables automatically. Save.

VERIFY
Hard-refresh your storefront (Ctrl+Shift+R / Cmd+Shift+R). Headings
and body now render in the kit fonts. In DevTools → Network, filter
.woff2 — the files should load from your own domain (not a third
party). That's the whole install.
