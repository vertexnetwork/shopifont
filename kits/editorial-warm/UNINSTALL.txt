UNINSTALL — Editorial Warm
================================================

Nothing in this install is permanent or stateful. To remove it
completely:

1. In your theme code editor, open the file where you pasted the
   @font-face + CSS-variable blocks (assets/base.css for every free
   Shopify theme). Delete both blocks. They are contiguous and the
   only thing the install added to that file.

2. Delete the .woff2 files you uploaded from the assets/ folder:
      lora-400.woff2
      lora-700.woff2
      open-sans-400.woff2
      open-sans-700.woff2

3. Hard-refresh your storefront (Ctrl+Shift+R / Cmd+Shift+R). Your
   theme's original fonts are back. There is no other state to clean
   up — the install was two CSS blocks and some font files, nothing
   more.
