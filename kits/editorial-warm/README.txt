SHOPIFY TYPOGRAPHY KIT — Editorial Warm
================================================

For: Editorial brands, artisan food, and long-copy stores
Fonts: Lora (headings) + Open Sans (body)
File budget: ~150KB total (Regular + Bold each, WOFF2)

WHY THIS PAIRING
Lora is a serif designed for body reading — calligraphic roots, comfortable at long lengths. Used in Bold as a heading face it brings editorial gravitas without the high-contrast drama of Playfair. Open Sans handles secondary body and UI text.

THE ONE MISTAKE TO AVOID
Lora wasn't built for huge display sizes. The kit uses Lora Bold (700) for headings and caps it at H1 — for oversized hero text, use the Premium Editorial or Luxury Classic kit instead.

WHAT'S IN THIS KIT
- fonts/DOWNLOAD.txt ....... exactly which font files to get (free) and how
- SPECIMEN.html ........... open in any browser to see the pairing
- install/<theme>/ ........ two copy-paste blocks for all 13 free Shopify themes
- LICENSE-AND-USAGE.txt ... you're cleared for commercial Shopify use; details inside
- UNINSTALL.txt ........... clean reversal in case you change your mind

QUICK START (~5 minutes)
1. Open fonts/DOWNLOAD.txt and get the font files (free, from Google Fonts).
2. Find your Shopify theme in install/ (e.g. install/dawn/).
3. Open INSTALL.txt in that folder — two blocks to paste, that's it.
4. Open SPECIMEN.html any time to preview what you're installing.

No account, no dependencies, nothing phones home. The install is pure
